Welcome to GitAIOps

GitAIOps is a ChatGPT plugin that allows you to manage your GitLab and GitHub CI/CD workflows directly from your conversation with ChatGPT. You can review merge requests, commit changes, find errors in pipelines, update MR descriptions and much more.

For more information, please visit our GitHub repository at https://github.com/mhdzumair/GitAIOps.
